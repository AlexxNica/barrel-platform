This application provides variants of key components from the Erlang/OTP libraries, adapted to run under PULSE and compiled with PULSE instrumentation. At present instrumented code is provided for
  • application
  • application_controller
  • application_master
  • application_starter
  • gen
  • gen_event
  • gen_fsm
  • gen_server
  • proc_lib
  • supervisor
Adding the include directive 
-include_lib("pulse_otp/include/pulse_otp.hrl").

to a file which is instrumented by PULSE will cause PULSE to replace calls to these OTP libraries with calls to the instrumented versions, which is normally all that is needed to enable client code to be tested under PULSE.
